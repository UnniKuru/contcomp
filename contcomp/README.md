ControlsComp
===============================

version number: 0.0.0
author: Unni Kurumbail

Overview
--------

Compares control algorithms for a given model.

Installation / Usage
--------------------

To install use pip:

    $ pip install contcomp


Or clone the repo:

    $ git clone https://github.com/UnniKuru/contcomp.git
    $ python setup.py install
